## 如何使用
当你的电脑安装过node环境（版本为8或以上），则下载后进入到该项目，只需运行以下命令即可 build 出 webpack 打包后的 mini-promise
```
npm install
npm run build
```

如果想体验 mini-promise，可以进入 examples 文件夹运行以下命令，就可以体验 mini-promise

```
node demo
```
